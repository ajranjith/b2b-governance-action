Install GRES B2B

Auto Install
Manual

Windows (PowerShell)

irm https://raw.githubusercontent.com/ajranjith/b2b-governance-action/main/templates/wizard/auto-install.ps1 | iex

Or download and double-click gres-b2b.exe from the latest release.

macOS / Linux

curl -sL https://raw.githubusercontent.com/ajranjith/b2b-governance-action/main/templates/wizard/auto-install.sh | bash
The installer will download the binary, verify checksums, and optionally run the setup wizard.

Manual Installation

  1. Download the binary from GitHub Releases
  2. Extract to a folder in your PATH
  3. Run gres-b2b --setup to configure MCP

MCP Configuration

After installation, add this to your MCP settings:

{ "mcpServers": { "gres-b2b": { "command": "gres-b2b", "args": [] } } }
